# Stand-Log-Tools
为 Stand 打造的日志实时查看工具

## 功能
+ 自动识别 log.txt 位置
+ 实时显示 Stand 日志
+ 不显示历史日志（缩短加载时间）
+ Windows 11 中文支持
+ 更好的中文字体
+ 自动适应高 DPI 显示器
+ 为不同类型的日志添加高亮
  + 黄：检测、重复次数
  + 红：崩溃、攻击、踢、事件
  + 蓝：（脚本）主机更替
  + 紫：Stand
  + 灰：时间戳
  + 白：聊天&其它
